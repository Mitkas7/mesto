# Проект: "Место"
#### Создан в рамках обучения по специальности "Веб разработчик" в [Яндекс Практикуме](https://praktikum.yandex.ru)


### Используемые технологии
* Flexbox
* Grid Layout
* CSS Animations
* JavaSctipt
* Методология БЭМ, включая файловую структуру БЭМ Nested
* Git
